# 💰 Change Maker Project

A simple C++ project based on the **Greedy Algorithm** concept from Discrete Mathematics.  
This project helps cashiers calculate the minimum number of notes and coins required to return change efficiently.

---

## 🚀 Features

- Calculates minimum currency notes and coins
- Uses Greedy Algorithm approach
- Simple console-based interface
- Fast and efficient calculations
- Beginner-friendly implementation

---

## 🧠 Algorithm Used

This project uses the **Greedy Algorithm**, where the program always selects the largest possible denomination first until the remaining amount becomes zero.

### Example

If the change amount is `786`, the program returns:

| Denomination | Quantity |
|-------------|----------|
| 500 | 1 |
| 100 | 2 |
| 50 | 1 |
| 20 | 1 |
| 10 | 1 |
| 5 | 1 |
| 1 | 1 |
